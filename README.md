# AA Wallet Solidity

## Setup 

```shell
npm install
```

## Deploy


### Set env

1. Copy config
    ```shell
    cp .env.example .env
    ```
2. modify environment variables
    ```
    ETHERSCAN_API_KEY=ABC123ABC123ABC123ABC123ABC123ABC1
    # your ethereum address private key
    PRIVATE_KEY=0xabc123abc123abc123abc123abc123abc123abc123abc123abc123abc123abc1
    ```
